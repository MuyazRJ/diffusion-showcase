The models

Three models, increasing complexity

Each stage introduces new architecture choices and conditioning mechanisms — from a compact baseline to a full latent text-conditioned system, with all forward noising and reverse sampling implemented from scratch.

Model 01
MNIST Diffusion
Baseline DDPM · U-Net
6.23 FID · N = 10,000 · test split

A compact U-Net trained to predict noise at each diffusion timestep. Uses residual blocks with GroupNorm and SiLU, sinusoidal timestep embeddings injected into every block, and a single self-attention layer at the 7×7 bottleneck for spatial coherence. The encoder path downsamples 28×28 → 14×14 → 7×7 with skip connections fused into the decoder.

Model 02
CIFAR-10 Diffusion
Improved DDPM · Class-conditional
9.60 FID · N = 10,000 · class-balanced

Scales to 128–256 channel widths with a 4-level encoder (32→16→8→4) and 8-head self-attention at the 8×8 and 16×16 scales. Class conditioning adds a learned 10×512 embedding to the timestep vector. Outputs both noise prediction and a learned variance head. EMA weights (λ=0.9999) used at inference; AMP reduces memory throughout training.

Model 03
Latent Text Diffusion
Text-conditioned · CUB-10
9.32 FID · N = 11,786 · train-set protocol

Runs diffusion in a frozen VAE's 4×32×32 latent space rather than 256×256 pixels. Frozen CLIP text embeddings (dim 768) condition the U-Net via cross-attention SpatialTransformer modules at the 16×16 and 8×8 scales. Classifier-free guidance enabled by dropping captions in 20% of training batches. DDIM sampler cuts inference to 50 steps (≈20× faster).

About this project

Progressive Diffusion Models for Image Generation

BSc Computer Science final year project by Mohammed Rahman (University of Manchester, 2026). The project implements diffusion training and sampling entirely from scratch in PyTorch — no pre-built diffusion frameworks — to evaluate how modern design choices affect generation quality as task complexity increases.

The structure is a controlled three-stage progression: a baseline DDPM on MNIST validates the end-to-end pipeline; the Improved DDPM on CIFAR-10 tests whether architectural and sampling upgrades produce a measurable quality jump; latent text diffusion on CUB-10 demonstrates prompt-guided generation at higher resolution under constrained compute.

All models were trained on a single desktop with an NVIDIA RTX 3060 (12 GB VRAM). Evaluation uses Fréchet Inception Distance (FID) as the primary metric alongside qualitative sample grids and denoising trajectories.
