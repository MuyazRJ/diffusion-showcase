# Diffusion Showcase

Interactive web demo for **Progressive Diffusion Models for Image Generation: From DDPMs to Latent Diffusion** — BSc Computer Science final year project, University of Manchester, 2026.

> Mohammed Rahman · 10971320

---

## Overview

A FastAPI + HTML website that lets you generate images live from three diffusion models trained from scratch in PyTorch, alongside the full backward denoising process as an animated GIF.

| Model | Dataset | Resolution | FID |
|---|---|---|---|
| Baseline DDPM | MNIST | 28 × 28 | 6.23 |
| Improved DDPM | CIFAR-10 | 32 × 32 | 9.60 |
| Latent text diffusion | CUB-200-2011 | 256 × 256 | 9.32 |

## Setup

### 1. Clone with submodules

```bash
git clone --recurse-submodules https://github.com/MuyazRJ/diffusion-showcase
cd diffusion-showcase
```

If you already cloned without submodules:

```bash
git submodule update --init --recursive
```

### 2. Install dependencies

```bash
pip install fastapi uvicorn pillow numpy aiofiles torch torchvision transformers diffusers
```

### 3. Add model weights

Place the trained `.pt` / `.pth` weight files in their respective folders (these are not tracked by git):

```
mnist_diffusion_unet/model_dict/mnist_unet.pth
cifar_10_diffusion/model_dict/final_ddpm.pt
latent_text_diffusion/  (path set in configs/model.yaml → training.load_checkpoint)
```
### 4. Run the server

```bash
py -3.12 -m uvicorn main:app --reload
```

Then open **http://127.0.0.1:8000** in your browser.

## Features

- **Landing page** — project overview, model cards with real FID scores and architecture details, results summary table
- **MNIST tab** — generates a handwritten digit with the full 1000-step backward process as a GIF
- **CIFAR-10 tab** — class-conditional generation with a dropdown for all 10 classes
- **CUB-200 tab** — text-guided bird generation with a free-text prompt and a choice of 50-step (fast) or 1000-step (higher quality) DDIM sampling
- **Hero animation** — cycles through real model outputs at t=1000, t=500 and t=0 side by side
